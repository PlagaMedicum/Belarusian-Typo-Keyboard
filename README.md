## Усталяванне

Для ўсталявання раскладцы вам трэба запусціць тэрмінал у гэтай дырэкторыі і запусціць наступную каманду:

```bash
chmod +x ./install.sh
./install.sh install
```

## Выдаленне

Калі ў вас узніклі праблемы, вы можаце вярнуць канфігурацыйныя файлы ў былы стан камандай

```bash
./install.sh backup
```
